## 📚 Overwatch Hero Selector
### 📖 개요
박영웅 강사님 스타벅스 사이트 만들기 강의를 수강 중 오버워치 히어로 선택창을 구현해보는 실습 수업이 있어, 강사님이 만드는 과정을 보기 전에 HTML과 CSS 연습을 위해 오버워치 히어로 선택창을 스스로 연습해보았다.
